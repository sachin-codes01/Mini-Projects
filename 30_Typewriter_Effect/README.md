# ⌨️ React Typewriter Effect

A clean and minimal **Typewriter Effect** implementation built with **React** using **Hooks** and **CSS animations**.

---

## 📸 Screenshot

![Typewriter Effect](public/tying%20animation.png)

---

## 🚀 Features

* ⌨️ Smooth character-by-character typewriter animation
* 💡 Built with native React Hooks — no external libraries
* 🟢 Terminal-style green-on-dark theme with monospace font
* ✨ Blinking cursor effect using pure CSS animation

---

## 🛠️ Technologies Used

* React
* JavaScript (ES6+)
* CSS3
* Vite

---

## 💡 Key Concepts Used

* React Hooks (**useState**, **useEffect**, **useRef**)
* **setInterval** for character-by-character text reveal
* **useRef** for mutable interval and index tracking without re-renders
* CSS **@keyframes** animation for blinking cursor effect
